# SEEKR Goal Status

Last updated: 2026-05-10.

## Objective

Finish SEEKR as a plug-and-play internal-alpha system: a local-first GCS with AI implemented through the local Ollama operator path, simulator/SITL flight core, replay/evidence flow, read-only drone integration, startup preflight, and review bundle evidence so an operator can install, start, and validate the local system with minimal setup. This remains an internal demo and audit system, not production flight firmware.

The aircraft safety rule is still absolute: real MAVLink, ROS 2, PX4, ArduPilot, geofence, mission upload, mode-change, arm, takeoff, land, return-home, terminate, and waypoint command paths stay blocked outside simulator/SITL transports.
