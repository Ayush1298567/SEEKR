# SEEKR Fleet Operations Manual

Created: 2026-05-04

## Required Fleet Fields

Every drone record must track:

- Aircraft ID and name.
- Firmware/software version.
- Remote ID status where applicable.
- Maintenance state.
- Flight hours.
- Last preflight check.
- Battery packs and cycles.
- Payload configuration.
- Command authority level.
- Emergency state.

## Preflight Minimums

- Confirm operating authority and airspace.
- Confirm Remote ID compliance where applicable.
- Confirm FC failsafe configuration.
- Confirm independent emergency/RC path.
- Confirm battery reserve.
- Confirm GCS/replay logging enabled.
- Confirm no real command class outside approved safety case.
